Features

Built for the terminal.
Not for the browser.

No Electron. No web UI. No config files. Just your terminal and an API key.

12 AI Providers

Anthropic, OpenAI, DeepSeek, Groq, Mistral, Together, Fireworks, Cerebras, OpenRouter, MiniMax — plus OpenAI-compatible and Anthropic-compatible endpoints. Switch mid-session.

9 Native Tools + ask_user

File read/write/patch, shell execution, semantic grep, glob search, directory listing, web search, diff review — plus interactive ask_user prompts with predefined options.

Self-Healing Loops

When a shell command fails or a file write is rejected, the agent reads the error, fixes the approach, and retries — up to 5 automatic recovery cycles per call.

Session Persistence

Auto-saved after every REPL interaction. Resume any session with /resume. Named sessions. Keeps last 10 auto-sessions with full conversation history.

Git & GitHub

Diff review, AI-generated commit messages, branch management, undo commits, full GitHub PR creation with changelogs — all from within the REPL.

Project Intelligence

Auto-detects 15 project types — Bun, Node, Rust, Go, Python, Elixir, .NET, Swift, PHP, Deno, C/C++, Gradle, Maven, and more. Suggests correct test/build commands.

Beautiful Terminal UX

40+ formatting helpers, Unicode tool glyphs, boxed sections, clean tables, color-coded diffs, animated spinners, visual context progress bar.

File Watcher

Watch files for changes with /watch and auto-run tests on every save. Continuous feedback loop during development.

Ecosystem

Multi-provider by design

Not locked into any single AI. Switch providers mid-session with /provider.

Quickstart

Up and running in 30 seconds

No install required. Just Bun and an API key.

Bun (recommended) 
# install Bun (if you don't have it)
curl -fsSL https://bun.sh/install | bash

# run directly - no global install needed
bunx aura-core

# set your API key
export ANTHROPIC_API_KEY="sk-ant-..."

# start the REPL
aura

# or one-shot mode
aura "Fix all TODOs in the codebase"

# open command picker
# type / in the REPL
Build from source 
# clone the repo
git clone https://github.com/rut123321/aura-core.git
cd aura-core

# install dependencies
bun install

# build binary (produces aura.exe)
bun run build

# run directly
./aura

# set your API key
export OPENAI_API_KEY="sk-..."
Workflow

From idea to pull request

1
Start a session
aura opens the REPL. Pick a provider and model. Sessions auto-save after every turn.
2
Describe the task
Plain English. "Add user authentication", "Refactor the auth module", "Fix the login bug".
3
Review & iterate
Agent shows diffs, asks before destructive operations. Use /review for AI code review.
4
Ship it
/commit for AI-generated commit messages or /pr for full GitHub PRs with changelogs.
